Irrepressible Minions

Personal Assistant

This application is designed to manage contacts, phone numbers, notes, and tags as well as to to manage files, their descriptions, and user associations. It provides an intuitive user interface for efficiently managing personal or professional contacts, ensuring all relevant information is easily accessible and organized. Moreover, it provides an intuitive user interface for efficiently managing personal or professional files, ensuring all relevant information is easily accessible and organized

Main Features


1. Contact Management

  • Add, update, and delete contacts
  • Each contact can have multiple phone numbers and notes associated with it
  • Search contacts by name, phone number, email, birthday, and tags

2. Phone Number Management

  • Add, update, and delete phone numbers
  • Normalization of phone numbers to ensure consistency

3. Note Management

  • Add and update notes associated with contacts
  • Delete notes while keeping the contact information intact

4. Tag Management

  • Add, update, and delete tags
  • Search contacts by tags to quickly find contacts associated with specific tags

5. Search Functionality

  • Comprehensive search options, including searching by name, phone number, email, birthday, and tags
  • View upcoming birthdays for contacts within the next week

6. User Authentication

  • Secure login required for accessing and managing contact information
  • Each user can only access and manage their own contacts

7. Pagination and Organization

  • Paginated views for managing large lists of contacts, phone numbers, notes, and tags

8. RSS Feed Integration

  • Display the latest news articles from RSS feeds
  • Paginated view for browsing through news articles

9. File Management

  • Add, update, and delete files: Users can upload new files, modify existing ones, and remove files they no longer need
  • File validation: Ensures that files meet size and type restrictions, with a maximum file size of 100 MB and support for various file types (e.g., images, PDFs, Word documents, Excel spreadsheets, PowerPoint presentations)

10. File Description

  • Description management: Each file can have a description associated with it, which can be updated as needed

11. User Association

  • User-specific file storage: Files are associated with individual users, ensuring that each user can only access and manage their own files

12. Edit and Remove Functionality

  • Edit file descriptions: Users can update the descriptions of their files as needed
  • Remove files: Users can delete files, with the file being removed from both the database and the file system
